

\chapter{FAQ}
\label{sec:faq}

This chapter answers some frequently asked questions.

\begin{enumerate}[Q 1.]
  \item I'd like to add a new file in Macsim. What should I do? 

\begin{enumerate}[1.]
\item First add files in src directory.
\item Add an entry in macsim/trunk/.obj/Makefile.am.
\begin{Verbatim}
  all_sources = \
    ... \
    ../src/new_file.cc
\end{Verbatim}
\end{enumerate}

\end{enumerate}
